
### 方法B: Supabase Dashboardで手動反映する

必要な操作:

1. SQL Editorで `supabase/migrations/202605310001_learning_records.sql` を実行する。
2. SQL Editorで `supabase/migrations/202606200001_registration_number_login.sql` を実行する。
3. Edge Function `lingopilot-auth` をデプロイする。
4. Function secretsを設定する。
5. 管理画面でテストユーザーを登録する。
6. 登録番号とメールアドレスでログイン確認する。

## 注意

- 秘密情報はリポジトリに保存しない。
- Access Token、DB Password、Service Role Keyはチャットやドキュメントに貼らない。
- PowerShellの環境変数として一時的に設定してから実行するのが安全。
